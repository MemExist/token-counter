# Global AI Token Counter / 全球 AI Token 计数器

一个轻量级、纯前端的 AI Token 计数与文本分析工具。  
支持全球主流 AI 模型的 Token 估算、上下文窗口占用、输入/输出费用预估，以及 AI 文本分析报告。

A lightweight, frontend-only AI Token counter and text analysis tool.  
It supports token estimation, context window usage, input/output cost estimation, and AI-powered text analysis for global AI models.

## Features / 功能

- 精确 Token 计数
- 字符数、词数、行数统计
- AI 模型上下文窗口占用显示
- 输入费用、输出费用、总费用预估
- 支持全球主流 AI 模型列表
- 支持深色 / 浅色主题切换
- 纯 HTML 单文件，无需后端
- 可直接部署到 GitHub Pages

## Deploy / 部署

This project is a static single-page web app.  
You can deploy it directly with GitHub Pages.

1. Rename the main file to `index.html`
2. Upload it to your GitHub repository
3. Open repository `Settings`
4. Go to `Pages`
5. Select `Deploy from a branch`
6. Choose `main` and `/root`
7. Save and wait for deployment
